Simple system for creating quizzes

Installing
=========

* Install pip (http://pip.readthedocs.org/en/latest/installing.html)
* Install requirements
```
pip install -r requirements.txt
```
* $ cd sources
* Initialize database
```
$ ./manage.py syncdb
```
* Run
```
$ ./manage.py runserver
```
* Open browser and go to
```
http://localhost:8000
```
